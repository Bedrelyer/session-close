## Body

```
Hi HN,

I built session-close, an open-source Agent Skill for Cursor, Claude Code, and Codex.

The idea comes from a pattern I saw going around: end every AI session with two questions:

1. What should I have asked you?
2. What am I missing?

This skill automates that into a structured handoff file in your repo (.session-close/handoffs/) so your next session can resume without re-explaining context. Inline mode answers Q1/Q2 in one turn — no extra chat round.

Install:
  git clone https://github.com/Bedrelyer/session-close.git && cd session-close && ./install.sh cursor

Then say "wrap up" or @session-close.

GitHub: https://github.com/Bedrelyer/session-close

Would love feedback on the handoff format and reflection tiers (L0–L4). MIT licensed.
```
